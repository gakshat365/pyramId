# Vought International Pyramid Scheme Manager

This project is a database management system for "Vought International," managing a multi-universe recruitment structure. It includes a Python Command Line Interface (CLI) for interacting with a MySQL database.

## Prerequisites

Before running the application, ensure you have the following installed on your system:

1.  **MySQL Server**: [Download MySQL Community Server](https://dev.mysql.com/downloads/mysql/)
2.  **Python 3.x**: [Download Python](https://www.python.org/downloads/)
3.  **MySQL Connector for Python**: This library is required for Python to talk to the database.

## Installation & Setup
